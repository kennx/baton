# M5StickS3 万能红外遥控器 (Baton)

一款基于 M5StickS3 的离线智能万能红外遥控器。内置多品牌电视及家电的红外特征码库，无需依赖网络或手机，即可通过简单的按键交互快速匹配并控制身边的家电设备。

**注意：** M5StickS3 硬件**没有内置红外接收器**，因此无法“学习”或“克隆”现有的实体遥控器信号。本项目通过内置的红外码库（发射专用的）实现万能遥控功能。

## 硬件需求

- [M5StickS3](https://docs.m5stack.com/en/core/M5StickS3)（ESP32-S3，8MB Flash + 8MB PSRAM）
- 内置红外发射管（GPIO46）
- 135×240 ST7789 屏幕
- 双按钮操作（主按键 BtnA，侧边按键 BtnB）

## 编译与烧录

```bash
# 编译
pio run -e m5stick-s3

# 烧录到设备
pio run -e m5stick-s3 --target upload

# 运行本地测试（验证红外码库）
pio test -e native
```

## 操作手势说明

系统深度优化了双按键交互逻辑，通过短按与长按的组合，实现完整的设备选择与控制：

| 手势 | 动作含义 |
|------|------|
| **短按** BtnA | 切换下一项 / 发射当前指令 |
| **长按** BtnA（>500ms） | 确认选择 / 切换控制指令 |
| **短按** BtnB | 返回上一级菜单 |

---

### 主菜单

开机后进入主菜单，显示四个核心选项：

| 菜单项（屏幕显示） | 中文含义 | 功能 |
|-------------------|----------|------|
| **Ctrl** | 控制 | 选择已保存的家电设备进行遥控 |
| **Match**| 匹配 | 从内置码库中寻找并添加新家电 |
| **Mgmt** | 管理 | 删除不再使用的设备配置 |
| **Set** | 设置 | 调整发射次数、亮度、蜂鸣器等 |

- **短按 BtnA** 切换菜单项，**长按 BtnA** 进入对应模式。

---

### Match（匹配模式 - 添加设备）

内置离线码库，无需连网即可找到您的家电。

- 进入后，**短按 BtnA** 浏览内置的家电品牌（如 Samsung TV, LG TV, Sony TV 等）。
- 选中您的品牌后，**长按 BtnA** 进入“电源测试”环节。
- 设备将自动向外发射该品牌的“电源”指令。如果您的家电成功关机/开机：
  - **长按 BtnA**：进入保存确认界面，再次长按即可将该设备加入您的遥控列表。
  - **短按 BtnB**：如果设备未响应，可返回上一步继续尝试。

---

### Ctrl（控制模式 - 万能遥控）

控制已保存的家电。

1. **选择设备**：
   - 进入后会看到已保存的设备列表。
   - **短按 BtnA** 循环选择设备。
   - **长按 BtnA** 选中当前设备，进入“指令面板”。
2. **控制设备（指令面板）**：
   - 屏幕将显示当前的待发指令（如 `[Power]`, `[Vol+]`）。
   - **短按 BtnA**：立即发射红外信号（屏幕提示 `SENDING...`）。
   - **长按 BtnA**：切换到下一个按键指令（如从 `Vol+` 切换到 `Vol-`）。
   - **短按 BtnB**：退出指令面板，返回设备选择列表。

---

### Mgmt（管理模式）

管理已保存的家电设备。

- **短按 BtnA** 选择一个已保存的设备。
- **长按 BtnA** 弹出删除确认框。
- 在弹窗中，短按切换 Cancel/Confirm，选中 Confirm 后长按彻底删除该设备配置。
- **短按 BtnB** 随时返回。

---

### Set（设置模式）

调整系统全局设置。

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| **Clear all**（清空所有） | — | 删除所有已保存的设备配置，需二次确认。 |
| **Repeat**（发射重复） | 2 | 每次按下按键时发射红外代码的重复次数（1→5 循环）。 |
| **Brightness**（亮度） | 80% | 屏幕亮度百分比（20→100 循环）。 |
| **Buzzer**（蜂鸣器） | ON | 是否开启操作时的蜂鸣提示音。 |
| **About**（关于） | — | 显示固件版本、已存设备数量、存储占用容量。 |

- **短按 BtnA** 切换设置项，**长按 BtnA** 执行更改或进入子菜单。

---

## 存储架构

- 系统底层采用 LittleFS，最多可保存 **20 个不同的家电配置 (Device Profile)**。
- 采用 JSON 格式持久化存储，重启不丢失数据。
- 架构全面解耦，所有的协议和按键映射均静态预定义在 `DeviceDatabase` 中。

## 注意事项

- **红外需要视线传输**：发射时请确保 M5StickS3 的红外灯（顶部）对准并直视目标设备，无物理遮挡。
- **匹配失败**：如果内置库中未能匹配到您的特殊设备，未来我们计划引入 Web 页面同步功能，让您可以从云端下载极客红外数据库 (如 Flipper IRDB) 中的特定配置。
